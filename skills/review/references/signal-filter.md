# Signal Filter

Use this reference to keep `review` focused on high-signal issues only.

Surface only issues that matter soon:

- bugs
- regressions
- missing verification
- scope drift
- maintainability problems likely to hurt the next change

Skip:

- style nits
- speculative refactors
- "could" suggestions with no concrete risk

## Rules

- Use this reference only to decide whether an issue is worth surfacing.
- Do not use this reference to change `review` status meanings or output shape.
