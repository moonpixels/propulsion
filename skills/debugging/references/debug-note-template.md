# Debug Note Template

Use this for non-trivial investigations.

Write `docs/propulsion/{yyyymmdd}-{issue-name}/debug.md`.

```md
# Debug Note: <issue>

## Reproduction

- exact steps

## Evidence

- errors, traces, logs, commands

## Hypotheses Tried

- hypothesis
    - result

## Root Cause

- grounded diagnosis

## Fix Direction

- next change to make in `execution`
```
